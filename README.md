# Event Manager Service

## Описание
Event Manager — backend-микросервис для управления мероприятиями.
Сервис предоставляет REST API для создания, обновления и получения информации о событиях.
Проект реализован как часть event-driven архитектуры и публикует доменные события при изменении состояния мероприятий.

Проект был разработан с упором на реальные бизнес-сценарии и микросервисный подход.

---

## Функциональность
- Создание мероприятия
- Получение списка мероприятий
- Получение мероприятия по id
- Обновление данных мероприятия
- Публикация событий о создании/изменении мероприятия в Kafka

- Событийно-ориентированный подход (event-driven)

---

## Взаимодействие сервисов
При создании или изменении мероприятия:
1. Event Manager сохраняет данные в базе
2. Публикует событие в Kafka (например `EVENT_CREATED`)
3. Notification-service подписывается на события и обрабатывает их
